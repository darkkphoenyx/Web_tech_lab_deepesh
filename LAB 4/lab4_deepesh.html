Inline CSS

CSS is used to add styling to HTML elements. Here we demonstrate the concept of Inline CSS. CSS properties are added to start tag of HTML.

Internal CSS

Internal CSS are used to style the HTML elements within same HTML page. Inside the header section we declare our CSS properties in style tag. Here we are using class selector.

Representing ID selector

Id selectors are used to select and HTML element with corresponding ID in the HTML. We use # operator for ID selector.

External CSS

External CSS is used to style HTML elements outside of HTML document. For this we link our CSS file using link tag.
